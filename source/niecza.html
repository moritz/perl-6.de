[% readvar title %]

Niecza begann als Projekt, um herauszufinden, wie man einen optimierenden Compiler für Perl 6 schreiben könne. Inzwischen ist es neben Rakudo der zweite gut benutzbare Perl 6-Compiler.

Niecza generiert für vielen Perl 6-Code deutlich schnelleren Code als Rakudo, unterstützt aber generell ein paar weniger Features als Rakudo.
