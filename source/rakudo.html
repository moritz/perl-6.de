Rakudo Perl 6 ist der am aktivsten entwickelte Perl 6 compiler. Jeden Monat wird eine neue Version des Compilers freigegeben, und im monatlichen bis vierteljährlichen Takt veröffentlichen die Entwickler eine neue Version der Distribution Rakudo Star, die den Compiler, Dokumentation, einen Debugger und einige Module mit einer einzigen Installation bereitstellt.

Rakudo Star kommt ausserdem mit dem Modul-Installierer Panda.

Rakudo Benutzt den Compiler NQP (Not Quite Perl 6) für den Bootstrapping-Prozess. NQP wiederum basiert auf der Parrot Virtual Machine -- es gibt aber auch Pläne, NQP und Rakudo auf die Java Virtual Machine zu portieren.
