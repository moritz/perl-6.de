Rakudo Perl 6 ist im Moment (Stand: Januar 2014) der einzige aktiv entwickelte Perl 6 Compiler. Jeden Monat wird eine neue Version des Compilers freigegeben, und im monatlichen bis vierteljährlichen Takt veröffentlichen die Entwickler eine neue Version der Distribution Rakudo Star, die den Compiler, Dokumentation, einen Debugger und einige Module mit einer einzigen Installation bereitstellt.

Rakudo Star kommt ausserdem mit dem Modul-Installierer Panda.

Rakudo Benutzt den Compiler NQP (Not Quite Perl 6) für den Bootstrapping-Prozess. NQP wiederum kann auf drei Virtuelle Maschinen aufsetzen: auf der Parrot Virtual Machine, Java Virtual Machien und auf MoarVM.
